Instalación desde repositorio GitHub

Con el fin de obtener el código fuente del framework, es posible usar las opciones de clonación de repositorio remoto que entrega GitHub:

  1. HTTPS:
    git clone https://github.com/Javernaver/TSP-Framework.git
  2. SSH:
    git clone git@github.com:Javernaver/TSP-Framework.git
  3. GitHub CLI:
    gh repo clone Javernaver/TSP-Framework

Como opcion adicional, existe la posibilidad de descargar el código como archivo comprimido ZIP:

Haga click en la imagen para descargar

Prerrequisitos

Una vez obtenido el código de manera local, es necesario instalar los prerrequisitos con el fin de que el framework trabaje de forma correcta. Para ello se debe ejecutar el siguiente comando en la carpeta que contiene los archivos del software:

  1. pip install -r requirements.txt
Nota:

En caso de no tener instalado el instalador de paquetes de Python llamado pip, se puede instalar con el comando de Python

  1. py -m ensurepip --upgrade

Ejecución

Ya con los prerrequisitos instalados, se debe ejecutar el programa con Python en el directorio del framework indicando la instancia a analizar, un ejemplo a continuación:

  1. python tspf.py --instance instances/burma14.tsp
