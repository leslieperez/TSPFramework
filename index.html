TSP Framework

El problema del vendedor viajero (TSP) es un problema de optimización en el que se debe encontrar una ruta óptima que visite un conjunto de ubicaciones o ciudades para luego volver al punto de partida. Esta ruta debe minimizar los costos o la distancia recorrida.

El software TSP-Framework tiene como enfoque el solucionar instancias de este tipo de problema gracias al uso de las metaheurísticas de Simulated Annealing y Algoritmos Genéticos.
