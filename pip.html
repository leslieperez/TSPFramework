Instalación desde Paquete de Instalador PIP (Windows)

La instalación desde el servicio de instalador de paquetes de Python denominado pip, permite disponer del framework sin la necesidad de descargar el código fuente ni aplicar los prerrequisitos fundamentales.

Existen dos comandos que permiten realizar el proceso de instalación (seleccionar solo una):

  1. PyPi:
    pip install TSP-Framework
  2. GitHub:
    pip install git+https://github.com/Javernaver/TSP-Framework.git
Nota:

En caso de no tener instalado el instalador de paquetes de Python llamado pip, se puede instalar con el comando de Python

  1. py -m ensurepip --upgrade

Una vez instalado tendra acceso al comando tspf el cual podra ser utilizado desde cualquier directorio. Para ello, debe tener el directorio en las variables de entorno o PATH del sistema operativo.

Configuración de variables de entorno

Con el fin de ejecutar el comando tspf sin la necesidad de usar el entorno de Python en la consola de comandos, es necesario agregar un path en las variables de entorno del sistema.

http://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/tsp/

  1. Buscar “Editar las variables de entorno del sistema”

  2. En la ventana de propiedades del sistema, seleccionar el botón “Variables de entorno…”

  3. Buscar la variable “path” y se edita

  4. Agregar la siguiente dirección en un path nuevo, modificando el nombre de usuario y la versión de Python instalada:

    C:\Users\[Usuario]\AppData\Local\Packages\PythonSoftwareFoundation.Python.3.9_qbz5n2kfra8p0\LocalCache\local-packages\Python39\Scripts

    *Aplicable para Python 3.9

Ejecución

Una vez instalado el software y configurado las variables de entorno, el programa podrá ejecutarse desde cualquier directorio usando el comando tspf. Para ello es necesario indicar la instancia a evaluar en formato tsp, por ejemplo:

  1. tspf --instance archivo.tsp
Nota:

Es necesario tener alguna instancia en formato TSP, en caso de usar el mismo directorio descrito anteriormente, la instancia debe ubicarse en la siguiente dirección:

  1. C:\Users\[Usuario]\AppData\Local\Packages\PythonSoftwareFoundation.Python.3.9_qbz5n2kfra8p0\LocalCache\local-packages\Python39\Scripts

    2. *Aplicable para Python 3.9
